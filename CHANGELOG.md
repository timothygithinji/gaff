# Changelog

## 1.0.0

First public release.

Gaff is a collaborative rental-hunting tool for two people sharing one search:
cross-portal scraping (Rightmove, Zoopla, OpenRent) with de-duplication, AI and
public-data enrichment (commute, council tax, broadband, EPC, amenities), a
shared swipe-and-shortlist flow with a mutual-match pipeline, and a daily digest
email.

Built and used for a single real house hunt, then open-sourced and parameterised
so anyone can deploy their own instance. See the [README](README.md) and
[docs/DEPLOY.md](docs/DEPLOY.md) to get started.
